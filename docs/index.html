🎯 But du projet

AgriFlow est une application desktop JavaFX qui digitalise et optimise la gestion agricole en Tunisie. Elle permet aux agriculteurs, experts et administrateurs de gérer les parcelles, cultures, diagnostics, marketplace, irrigation et bien plus — le tout assisté par l'intelligence artificielle (Google Gemini).

⚙️ Instructions d'installation

  1. Installer Java 17+ et MySQL Server
  2. Importer le fichier agriflow9.sql dans phpMyAdmin
  3. Télécharger la release ci-dessous
  4. Ouvrir le projet dans IntelliJ IDEA
  5. Lancer la classe principale : mains.MainFX
